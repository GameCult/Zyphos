# Biosphere

The Zyphos biosphere is organized here by trophic role, with each organism treated as a behavioral system rather than a decorative analogue. Trophic level describes the main energy relationship. It does not erase memory contracts, route obligations, reproductive bargains, or political agency.

## Modeling Rule

For NPC behavior, each species page names five usable pressures: appetite, memory, social contract, threat response, and bargaining failure. A creature on Zyphos does not need humanlike speech to have leverage. It only needs something another lifeform cannot safely ignore.
